# 智能体接入说明

## 接入目标

你们只需要按要求调用页面提供的一个方法：

- `window.startAgentTiming`

## 接入步骤

### 1. 开始执行前先调用

在智能体开始处理任务前，必须先调用：

```js
window.startAgentTiming({
  attachmentName: 'invoice-001.pdf',
  prompt: '请从附件中提取转账信息并填入页面表单'
})
```

要求：

- `prompt` 必填，页面输入的prompt
- `attachmentName` 选填，传当前解析文件名
- 调用一次即可，后续不需要再调用结束方法
- 初始化不会上送，只有最终提交时才会上送一次

### 2. 正常完成填表并提交

智能体完成解析后，按页面原有方式填写并提交即可。

要求：

- 不需要团队额外调用结束方法
- 页面会在提交时自动结束计时并上送
- 最终只会上送一次
- 本次记录依靠文件名和提交内容识别

## 接入要求

- 必须先调用开始方法，再进行页面操作和提交
- 不允许绕开页面原有提交流程
- 提交时页面会自动带上解析结果和计时信息

## 强制约束

以下文件属于计时与统计代码，禁止修改：

- `src/utils/agentTimingBridge.js`

如发现擅自修改计时相关代码、绕过计时流程、删除计时上送逻辑，直接取消 POC 资格。
